# 3D_Solar_System
 [과거 프로젝트]OpenGL 3D Solar System View Implementation

# 프로젝트 설명
- 2013년도에 진행한 OpenGL 3D Solar System 구현 프로젝트
- 태양계의 공전, 자전 데이터를 비슷하게 시뮬레이션 해서 볼 수 있는 프로젝트
- OpenGL 3D Select 구현을 통해 선택한 천체의 정보를 알 수 있도록 하는 기능 구현
